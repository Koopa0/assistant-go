# Assistant 智能開發助手 - 文檔中心

## 🤝 如何貢獻

### 文檔改進
如果您發現文檔中的錯誤或需要改進的地方：

1. **小幅修正**: 直接提交 Pull Request
2. **大幅變更**: 先開 Issue 討論
3. **新增文檔**: 遵循現有文檔的結構和格式標準

### 文檔更新流程
1. 功能開發時同步更新相關文檔
2. API 變更時更新 API 文檔
3. 架構變更時更新架構文檔
4. Release 前檢查所有文檔同步性
